## 📌 Descrição

Este projeto contém a implementação de algoritmos em Java voltados para o desenvolvimento do raciocínio lógico e aplicação de estruturas condicionais, operadores matemáticos e geração de números aleatórios.

Os exercícios abordam situações práticas como cálculos matemáticos, sistemas de troco, equações e simulações computacionais.

---

## 🧠 Conteúdos abordados

* Entrada de dados com `Scanner`
* Estruturas condicionais (`if`, `else`, `switch`)
* Operadores matemáticos
* Funções da classe `Math`
* Geração de números aleatórios (`Random`)
* Lógica de programação

## 🚀 Exercícios

### 🔢 Exercício 1 — Maior, menor e média

O programa lê três números e exibe:

* O maior valor
* O menor valor
* A média aritmética

---

### 💰 Exercício 2 — Máquina de troco

Simula uma máquina de venda automática que:

* Recebe valor da compra e valor pago
* Verifica se o pagamento é suficiente
* Calcula o troco
* Informa a menor quantidade de notas possíveis

Notas disponíveis:

```id="l77e2x"
R$ 50, R$ 20, R$ 10, R$ 5, R$ 2 e R$ 1
```

---

### 📊 Exercício 3 — Equação do segundo grau

Resolve uma equação do tipo:

```id="p2ngl3"
ax² + bx + c = 0
```

O algoritmo:

* Valida os coeficientes
* Calcula o discriminante (Δ)
* Informa o tipo de solução:

  * Sem raízes reais
  * Raízes iguais
  * Raízes diferentes

---

### 🔵 Exercício 4 — Círculo e esfera

O programa executa diferentes operações com base em um código:

* 1 → Perímetro do círculo
* 2 → Área do círculo
* 3 → Volume da esfera

Caso o código seja inválido, o programa exibe uma mensagem de erro.

---

### 🧮 Exercício 5 — Calculadora

O programa lê:

* Dois números reais
* Um operador matemático (`+`, `-`, `*`, `/`, `^`)

E retorna o resultado da operação.
Caso o operador seja inválido, exibe erro.

---

### 🎲 Exercício 6 — Número aleatório

O programa:

* Lê dois números inteiros
* Determina automaticamente o menor e o maior
* Gera um número aleatório dentro do intervalo
* Informa se o número é par ou ímpar
